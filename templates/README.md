# Templates

このディレクトリは、提案書、メール、見積、ヒアリング、報告書などの雛形を管理する場所である。

雛形は文章を固定するためではなく、毎回の判断漏れを減らし、HACTACらしい品質を安定させるために使う。

## 今後作成する雛形

- 初回ヒアリングシート
- 提案書構成
- 見積前確認リスト
- 学習提案
- クライアントプロファイル
- メール文面
- 納品前チェックリスト
- 振り返りシート

## 運用方法

雛形は実案件で使いながら更新する。

定型文を増やしすぎず、確認すべき項目と判断の観点を残すことを優先する。

## メタデータ方針

テンプレートは、用途、機密性、承認状態を明記して管理する。

未検証の雛形案は `learning/` に置き、実案件で検証後に正式テンプレートへ反映する。

## 現在のテンプレート

- `templates/LEARNING_PROPOSAL.md`: 学習提案の雛形
- `templates/CLIENT_PROFILE.md`: クライアントプロファイルの雛形

## 要確認

- 既存の提案書、見積書、メール文面の有無
- 社外向けにそのまま使える文体
- 案件種別ごとに分ける必要がある雛形
