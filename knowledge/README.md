# Knowledge

このディレクトリは、HACTACの実務に関係する技術、業界、地域、教育、事業開発などの知識を整理する場所である。

単なるメモではなく、提案や制作判断に接続できる知識として管理する。

## 扱う領域

- WEB制作
- サーバー、業務自動化、AI活用
- n8n、NAS、セルフホスト運用
- ブランディング、デザイン、マーケティング
- 地域活性、教育、コミュニティ
- 事業開発、補助金、入札、契約

## 運用方法

知識は、出典、前提、使いどころをできるだけ明記する。

変化しやすい情報は、確認日や要再確認の条件を残す。

## メタデータ方針

技術・業界知識は、確認日、出典、変化しやすさをできるだけ残す。

未検証の知識や提案は `learning/` に置き、正式知識と混在させない。

## 要確認

- 優先的に蓄積する知識領域
- 社内向けメモと外部公開可能な情報の分け方
- 出典管理のルール
